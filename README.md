# Medical AI Assistant

## Project Documentation

### Overview
The Medical AI Assistant is designed to help users with medical-related queries, providing information and assistance using artificial intelligence.

### Features
- Symptom Checker
- Medical Information Retrieval
- Drug Interaction Checker
- Health Tips and Recommendations

### Installation
To install the Medical AI Assistant, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/drsandeepgupte-create/medical-ai-assistant.git
   ```
2. Navigate to the project directory:
   ```bash
   cd medical-ai-assistant
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
Run the application with:
```bash
python app.py
```
